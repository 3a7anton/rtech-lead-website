# RTech Lead

A modern, animated portfolio website built with React, TypeScript, and cutting-edge web technologies.

## Tech Stack

### Core
- **React 19.2.0** - UI library
- **TypeScript 5.9.3** - Type-safe JavaScript
- **Vite (Rolldown)** - Next-generation build tool with lightning-fast HMR

### Routing & State
- **React Router DOM 7.9.6** - Client-side routing

### Animation & Graphics
- **GSAP 3.13.0** - Professional-grade animation library
- **Three.js 0.181.2** - 3D graphics and WebGL rendering

### Build Tools
- **SWC** - Super-fast TypeScript/JavaScript compiler
- **ESLint** - Code quality and consistency

## Components

The project includes several reusable animated components:

- **FloatingLines** - Animated background elements with floating line effects
- **ScrollReveal** - Scroll-triggered reveal animations
- **ScrollFloat** - Floating elements that respond to scroll position
- **Navbar** - Responsive navigation bar with smooth transitions
- **Footer** - Site footer with social links and information
